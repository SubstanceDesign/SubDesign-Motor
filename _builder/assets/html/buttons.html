Default buttons

Default button is created with .button class applied to <a>, <button> or <input type=submit> tag.

Code

<a href="#" class="button">Default button</a>
<button class="button">Default button</button>
<input type="submit" class="button" value="Default button">

Button colors

Button have seven color modes: primary (default), neutral, secondary, info, success, warning and danger. The last six used by adding color name postfix to button class.

Code

<button class="button">Primary</button>
<button class="button-neutral">Neutral</button>
<button class="button-secondary">Secondary</button>
<button class="button-info">Information</button>
<button class="button-success">Success</button>
<button class="button-warning">Warning</button>
<button class="button-danger">Danger</button>

Button sizes

Button have three size modes excluding default. To change size add corresponding modifier .button--xs, .button--sm, .button--lg.

Code

<button class="button">Medium size</button>
<button class="button button--xs">Extrasmall size</button>
<button class="button button--sm">Small size</button>
<button class="button button--lg">Large size</button>

Demo

Button groups

Buttons can be united into seamless group by wrapping them with .button-group element.

Code

<div class="button-group">
  <button class="button">Success</button>
  <button class="button">Success</button>
  <button class="button">Success</button>
</div>
<div class="button-group">
  <button class="button-success">Success</button>
  <button class="button-warning">Warning</button>
  <button class="button-danger">Danger</button>
</div>
<div class="button-group">
  <a href="#" class="button-outline">Primary</a>
  <a href="#" class="button-outline">Primary</a>
  <a href="#" class="button-outline">Primary</a>
</div>

Semantic classes / Mixins

If you are a fan of semantic and clean HTML code, you may like to use your own class names for structure blocks. We designed mixins that can apply framework styles to your own classes.

Mixins list

Mixin Description Variables 
add_button_styles()
 Add necessary for any button settings and style resets.
You shouldn't use it with create_button-* mixins. It is included already		 - 
create_button-flat($background, $color, $border)
 Create flat button $background: color of button background. Required!

$color: color of button text. Required!

$border: color of button border. Required!		 
create_button-outline($border, $color, $border-width: 1px)
 Create outline button $border: color of button background. Required!

$color: color of button text. Required!

$border-width: border width. Optional!		 
create_button-3d($background, $color)
 Create 3D button $background: color of button background. Required!

$color: color of button text. Required!		 
create_button-gradient(
  $gradient-top, 
  $gradient-bottom, 
  $color, 
  $border-top, 
  $border-bottom
)
 Create flat button $gradient-top: top color of linear gradient. Required!

$gradient-bottom: bottom color of linear gradient. Required!

$color: color of button text. Required!

$border-top: color of button top border. Required!

$border-bottom: color of button bottom border. Required!

Examples:

SCSS

.custom-button-name {
  @include create_button-flat(#000, #fff, #ccc); // apply flat button styles
}

HTML

<button class="custom-button-name">Custom buttom</button>
Also you can include common input mixins. For example size mixin.

SCSS

.custom-button-name {
  @include create_button-outline(#ccc, #000); // apply outline button styles
  @include add_input_size(60px, 20px, 16px, 3px, 4px);
}

Setting buttons config

The framework has common input settings and individual button settings. Settings are in _source/config.scss file.

Settings list

Name Description Values
button_flat_border-width Border width of default flat button px
button_outline_border-width Border width of outline button px
button_gradient_border-width Border width of gradient button px
button_flat Enable default flat buttons true / false
button_outline Enable outline buttons true / false
button_3d Enable 3D buttons true / false
button_gradient Enable gradient buttons true / false
