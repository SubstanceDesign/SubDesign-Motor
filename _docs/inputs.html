Buttons

Default button is created with .button class applied to <a>, <button> or <input type=submit> tag.

Code

<a href="#" class="button">Default button</a>
<button class="button">Default button</button>
<input type="submit" class="button" value="Default button">

Demo

Default button

Button have seven color modes: primary (default), neutral, secondary, info, success, warning and danger. The last six used by adding color name postfix to button class.

Code

<button class="button">Primary</button>
<button class="button-neutral">Neutral</button>
<button class="button-secondary">Secondary</button>
<button class="button-info">Information</button>
<button class="button-success">Success</button>
<button class="button-warning">Warning</button>
<button class="button-danger">Danger</button>

Button have three size modes excluding default. To change size add corresponding modifier .button--xs, .button--sm, .button--lg.

Code

<button class="button">Medium size</button>
<button class="button button--xs">Extrasmall size</button>
<button class="button button--sm">Small size</button>
<button class="button button--lg">Large size</button>

Demo

Buttons can be united into seamless group by wrapping them with .button-group element.

Code

<div class="button-group">
  <button class="button">Success</button>
  <button class="button">Success</button>
  <button class="button">Success</button>
</div>
<div class="button-group">
  <button class="button-success">Success</button>
  <button class="button-warning">Warning</button>
  <button class="button-danger">Danger</button>
</div>
<div class="button-group">
  <a href="#" class="button-outline">Primary</a>
  <a href="#" class="button-outline">Primary</a>
  <a href="#" class="button-outline">Primary</a>
</div>
