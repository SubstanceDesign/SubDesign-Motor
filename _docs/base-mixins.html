Responsive mixins

Responsive mixins were designed to easily code in predefined screen breakpoints stystem. They accept breakpoints shortcodes as arguments.

Mixins list

Mixin Description Variables 
viewport-from($breakpoint)
 pick the range from breakpoint to infinity
useful for mobile first design		 $breakpoint: breakpoint shortcode name from $viewport_breakpoints map (by default mc, xs, sm, md, lg, xl). Required! 
viewport-while($breakpoint)
 pick the range from 0 to breakpoint inclusive $breakpoint: breakpoint shortcode name from $viewport_breakpoints map (by default mc, xs, sm, md, lg, xl). Required! 
viewport-from-to($from, $to)
 pick the range from one breakpoint to another $from, $to: breakpoint shortcode name from $viewport_breakpoints map (by default mc, xs, sm, md, lg, xl). Required! 
viewport-range($breakpoint)
 pick the range from one breakpoint to next greater breakpoint $breakpoint: breakpoint shortcode name from $viewport_breakpoints map (by default mc, xs, sm, md, lg, xl). Required!

Examples:

SCSS

.test-viewport-mixin {

  @include viewport-from(md) {
    content:'from md';
  }
}

output CSS

@media (min-width: 992px) {
  .test-viewport-mixin {
    content: 'from md';
  }
}

SCSS

.test-viewport-mixin {

  @include viewport-while(sm) {
    content:'while sm';
  }
}

output CSS

@media (max-width: 991px) {
  .test-viewport-mixin {
    content:'while sm';
  }
}

SCSS

.test-viewport-mixin {

  @include viewport-from-to(sm,lg) {
    content:'from sm to lg';
  }
}

output CSS

@media (min-width: 768px) and (max-width: 1199px) {
  .test-viewport-mixin {
    content:'from sm to lg';
  }
}

SCSS

.test-viewport-mixin {

  @include viewport-range(md) {
    content:'range md';
  }
}

output CSS

@media (min-width: 992px) and (max-width: 1199px) {
  .test-viewport-mixin {
    content:'range md';
  }
}

Crossbrowser mixins

Crossbrowser CSS mixins

Mixins list

Mixin Description Variables 
placeholder()
 set placeholder style -

Examples:

SCSS

textarea {

  @include placeholder() {
    color: #ccc;
  }
}

output CSS

textarea::-webkit-input-placeholder {  
  color: #ccc;
} 
textarea:-moz-placeholder {  
  color: #ccc;
} 
textarea::-moz-placeholder {
  color: #ccc;
} 
textarea:-ms-input-placeholder { 
  color: #ccc;
} 
textarea::input-placeholder, {   
  color: #ccc;
}
textarea.placeholder { 
  color: #ccc;
}

Other mixins

Mixins list

Mixin Description Variables 
clearfix()
 apply clearfix styles -

Examples:

SCSS

.my-row {
  @include clearfix();
}

output CSS

.my-row:before,
.my-row:after {
  content: '';
  display: table;
}
.my-row:after {
  clear: both;
}
