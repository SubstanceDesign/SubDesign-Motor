Default checkbox and radio

To use motor checkbox or radio create <label> tag with .checkbox or .radio class. Put input[type=checkbox] or input[type=radio] inside it, and create <span> tag next to input. You can place label text for the checkbox inside the span.

Code

<!-- checkbox -->
<label class="checkbox"><input type="checkbox" checked><span>Checkbox</span></label>

<!-- radio -->
<label class="radio"><input type="radio" name="radio-1" value="1" checked><span>Option 1</span></label>
<label class="radio"><input type="radio" name="radio-1" value="2"><span>Option 2</span></label>
<label class="radio"><input type="radio" name="radio-1" value="3"><span>Option 3</span></label>

Checkbox and radio styles

Besides default motor checkbox or radio style you can use filled style with .checkbox-filled and .radio-filled classes.

Code

<!-- checkbox -->
<label class="checkbox-filled"><input type="checkbox" checked><span>Checkbox</span></label>

<!-- radio -->
<label class="radio-filled"><input type="radio" name="radio-2" value="1" checked><span>Option 1</span></label>
<label class="radio-filled"><input type="radio" name="radio-2" value="2"><span>Option 2</span></label>
<label class="radio-filled"><input type="radio" name="radio-2" value="3"><span>Option 3</span></label>

Disabled state

For disabled state add ordinary disabled attribute to input and .disabled class to wrapper label.

Code

<label class="checkbox disabled"><input type="checkbox" checked disabled><span>Disabled</span></label>
<label class="checkbox disabled"><input type="checkbox" disabled><span>Disabled</span></label>
<label class="checkbox-filled disabled"><input type="checkbox" checked disabled><span>Disabled</span></label>
<label class="radio disabled"><input type="radio" name="radio-4" value="1" checked disabled><span>Disabled</span></label>
<label class="radio disabled"><input type="radio" name="radio-4" value="2" disabled><span>Disabled</span></label>
<label class="radio-filled disabled"><input type="radio" name="radio-5" value="3" checked disabled><span>Disabled</span></label>

Demo

Validation state

For validation states use .error, .warning and .success classes on label, input or label wrapper.

Code

<!-- Set error state on label -->
<label class="checkbox error"><input type="checkbox" checked><span>Error state</span></label>

<!-- Set error state on input -->
<label class="checkbox"><input class="error" type="checkbox" checked><span>Error state</span></label>

<!-- Set error state on wrapper -->
<div class="error"><label class="checkbox"><input type="checkbox" checked><span>Error state</span></label></div>

<!-- Set warning state -->
<label class="checkbox warning"><input type="checkbox" checked><span>Warning state</span></label>

<!-- Set success state -->
<label class="checkbox success"><input type="checkbox" checked><span>Success state</span></label>

Inline modificator

For the inline-block behavior use .checkbox--inline, .radio--inline classes.

Code

<label class="checkbox checkbox--inline"><input type="checkbox" checked><span>Option 1</span></label>
<label class="checkbox-filled checkbox--inline"><input type="checkbox"><span>Option 2</span></label>
<label class="radio radio--inline"><input type="radio" name="radio-6" checked><span>Option 1</span></label>
<label class="radio radio--inline"><input type="radio" name="radio-6"><span>Option 2</span></label>
<label class="radio radio--inline"><input type="radio" name="radio-6"><span>Option 3</span></label>

Semantic classes / Mixins

If you are a fan of semantic and clean HTML code, you may like to use your own class names for structure blocks. We designed mixins that can apply framework styles to your own classes.

Mixins list

Mixin Description Variables 
add_checkbox-radio_styles($type, $size)
 Add necessary for any checkbox and radio settings and style resets.
You shouldn't use it with create_* mixins. It is included already		 $type: 'checkbox' or 'radio'. Required!

$size: input size in px. Required!		 
create_checkbox($size, $background, $border, $inner)
 Create default checkbox. $size: input size in px. Required!

$background: background color. Required!

$border: border color. Required!

$inner: inner glyph color. Required! 

create_checkbox-filled($size, $background, $border, $border-active, $inner)
 Create filled checkbox. $size: input size in px. Required!

$background: background color. Required!

$border: border color. Required!

$border-active: checked border color. Required!

$inner: inner glyph color. Required! 

create_radio($size, $background, $border, $inner)
 Create default radio. $size: input size in px. Required!

$background: background color. Required!

$border: border color. Required!

$inner: inner glyph color. Required! 

create_radio-filled($size, $background, $border, $border-active, $inner)
 Create filled radio. $size: input size in px. Required!

$background: background color. Required!

$border: border color. Required!

$border-active: checked border color. Required!

$inner: inner glyph color. Required!

Examples:

SCSS

.custom-checkbox-name {
  @include create_checkbox(32px, #fff, #ccc, #000);
}

.custom-filled-radio-name {
  @include create_radio-filled(32px, #fff, #ccc, #000, #000);
}

HTML

<label class="custom-checkbox-name"><input type="checkbox"><span>Option</span></label>

<label class="custom-filled-radio-name"><input type="radio" name="custom-radio" value="1"><span>Option</span></label>

Setting checkbox and radio config

The framework has common input settings and individual checkbox and radio settings. Settings are in _source/config.scss file. Variables for the component starts from 'checkbox' word missing 'radio' part to reduce names length.

Settings list

Name Description Values
$checkbox_size Input side size px
$checkbox_border-width Input border width px
$checkbox_border-radius Input border radius px
$checkbox_label_font-size Input label font size px
$checkbox_label_line-height Input label line height float number
$checkbox_label_font-style Input label font style font-style css property values
$checkbox_label_font-weight Input label line height font-weight css property values
$checkbox_default Enable default checkboxes and radio true / false
$checkbox_filled Enable filled checkboxes and radio true / false
