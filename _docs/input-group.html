Input group part

You can mix text input in an inline group with any content. Place text input into element with .input-group class and add .input-group_part element. Now you can place any content inside input group part.

Code

<span class="input-group">
  <span class="input-group_part">Any content</span>
  <input type="text" class="text-input" placeholder="Default text input">
  <span class="input-group_part">Any content</span>
</span>

Input group appendix

To create stylized appendix add .input-group_appendix element.

Code

<span class="input-group input-group--collapse-outside">
  <span class="input-group_appendix">Price</span>
  <input type="text" class="text-input" placeholder="Default text input">
  <span class="input-group_appendix">$</span>
</span>

Input group button

To create input group with button add .input-group_part element and place any button inside. You can mix it with other parts and appendixes.

Code

<span class="input-group">
  <span class="input-group_part">
    <button class="button">Action</button>
  </span>
  <input type="text" class="text-input" placeholder="Default text input">
</span>

Input group collapse border

There is a common issue to remove double border in input group (one border from a button or an appendix and another from text input). To solve the problem use .input-group--collapse-outside and .input-group--collapse-inside modifiers. The first one remove borders of neighbor to text input elements. The second one remove borders of text input. To clearify the difference please see the example.

Code

<span class="input-group">
  <span class="input-group_part">
    <button class="button">Action</button>
  </span>
  <input type="text" class="text-input" placeholder="Default text input">
</span>

Input group sizes

Input groups have three size modes excluding default. To change size add corresponding modifier .input-group--xs, .input-group--sm, .input-group--lg.

Code

<span class="input-group input-group--xs input-group--collapse-outside">
  <span class="input-group_part">
    <button class="button">Action</button>
  </span>
  <input type="text" class="text-input" placeholder="Default text input">
  <span class="input-group_appendix">$</span>
</span>

Semantic classes / Mixins

If you are a fan of semantic and clean HTML code, you may like to use your own class names for structure blocks. We designed mixins that can apply framework styles to your own classes.

Mixins list

Mixin Description Variables 
create_input-group()
 Create input group - 
create_input-group--collapse-outside()
 Make input group to collapse outside borders - 
create_input-group--collapse-inside()
 Make input group to collapse inside borders - 
create_input-group_appendix(
  $padding-horizontal, 
  $font-size, 
  $background, 
  $color, 
  $border, 
  $border-width, 
  $border-radius)
 Create input group appendix $padding-horizontal: horizontal padding. Required!

$font-size: font size. Required!

$background: background color. Required!

$color: text color. Required!

$border: border color. Required!

$border-width: border width. Required!

$border-radius: border radius. Required!		 
add_input-group_appendix_size(
  $padding-horizontal, 
  $font-size, 
  $border-radius)
 Modify input group appendix size $padding-horizontal: horizontal padding. Required!

$font-size: font size. Required!

$border-radius: border radius. Required!

Examples:

SCSS

.custom-input-group-name {
  @include create_input-group();
  @include create_input-group--collapse-outside();
}

.custom-appendix-name {
  @include create_input-group_appendix(12px, 14px, #000, #fff, #ccc, 2px, 0);
}

HTML

<span class="custom-input-group-name">
  <span class="custom-appendix-name">Price</span>
  <input type="text" class="text-input" placeholder="Default text input">
</span>

Setting input group config

Settings are in _source/config.scss file.

Settings list

Name Description Values
$input-group_feature_collapse Enable collapse border feature true / false
$input-group_appendix_color_background Appendix background color
$input-group_appendix_color_border Appendix border color color
$input-group_appendix_color_text Appendix text color color
