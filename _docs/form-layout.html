Form layout

The basic form layout constists of optional .form-header, .form-footer and .from-group. Form header and form footer have larger margins from form content.

Form group can include any content, basicaly there are .form-label, one of input elements and help .form-text.

Code

<form action="#">
  <div class="form-header">
    <div class="h3">Form header</div>
  </div>
  <div class="form-group">
    <label class="form-label">Name</label>
    <input type="text" class="text-input" placeholder="Your name">
  </div>
  <div class="form-group">
    <label class="form-label">Email</label>
    <input type="text" class="text-input" placeholder="Your email">
  </div>
  <div class="form-group">
    <label class="switch">
      <input type="checkbox" checked>
      <span class="switch_lever"></span>
      <span class="switch_on">Show my email</span>
      <span class="switch_off">Hide my email</span>
    </label>
  </div>
  <div class="form-group">
    <label class="form-label">Password</label>
    <input type="password" class="text-input" placeholder="Your password">
    <div class="form-text">Please be sure to use only latin characters, numbers and symbol "_"</div>
  </div>
  <div class="form-group">
    <label class="checkbox"><input type="checkbox"><span>I accept the Terms and Conditions</span></label>
  </div>
  <div class="form-footer">
    <input type="submit" class="button" value="Submit">
  </div>
</form>

Inline form

To create one row form add .form-inline class to form tag.

Code

<form class="form-inline" action="#">
  <div class="form-group">
    <label class="form-label">Name</label>
    <input type="text" class="text-input" placeholder="Your name">
  </div>
  <div class="form-group">
    <label class="form-label">Password</label>
    <input type="password" class="text-input" placeholder="Your password">
  </div>
  <div class="form-group">
    <input type="submit" class="button" value="Submit">
  </div>
</form>

Horizontal form

To create horizontal form add .form-horizontal class to form tag and use grid columns inside each .form-group. Horizontal form class makes .form-group behave like a grid .row

Code

<form class="form-horizontal" action="#">
  <div class="form-group">
    <label class="form-label xs_col-2">Name</label>
    <div class="xs_col-10">
      <input type="text" class="text-input" placeholder="Your name">
    </div>
  </div>
  <div class="form-group">
    <label class="form-label xs_col-2">Email</label>
    <div class="xs_col-10">
      <input type="text" class="text-input" placeholder="Your email">
    </div>
  </div>
  <div class="form-group">
    <div class="xs_col-10 xs_col--offset-2">
      <label class="switch">
        <input type="checkbox" checked>
        <span class="switch_lever"></span>
        <span class="switch_on">Show my email</span>
        <span class="switch_off">Hide my email</span>
      </label>
    </div>
  </div>
  <div class="form-group">
    <label class="form-label xs_col-2">Password</label>
    <div class="xs_col-10">
      <input type="password" class="text-input" placeholder="Your password">
      <div class="form-text">Please be sure to use only latin characters, numbers and symbol "_"</div>
    </div>
  </div>
  <div class="form-group">
    <div class="xs_col-10 xs_col--offset-2">
      <label class="checkbox"><input type="checkbox"><span>I accept the Terms and Conditions</span></label>
    </div>
  </div>
  <div class="form-footer">
    <div class="xs_col-10 xs_col--offset-2">
      <input type="submit" class="button" value="Submit">
    </div>
  </div>
</form>

To make labels fit various size inputs use .form-label--xs, .form-label--sm, .form-label--lg modifiers.

Code

<form class="form-horizontal" action="#">
  <div class="form-group">
    <label class="form-label form-label--xs xs_col-2">Extrasmall</label>
    <div class="xs_col-10">
      <input type="text" class="text-input text-input--xs" placeholder="Your name">
    </div>
  </div>
  <div class="form-group">
    <label class="form-label form-label--sm xs_col-2">Small</label>
    <div class="xs_col-10">
      <input type="text" class="text-input text-input--sm" placeholder="Your name">
    </div>
  </div>
  <div class="form-group">
    <label class="form-label form-label--lg xs_col-2">Large</label>
    <div class="xs_col-10">
      <input type="text" class="text-input text-input--lg" placeholder="Your name">

</form>

Semantic classes / Mixins

If you are a fan of semantic and clean HTML code, you may like to use your own class names for structure blocks. We designed mixins that can apply framework styles to your own classes.

Mixins list

Mixin Description Variables 
create_form-label()
 Create form label - 
create_form-group()
 Create form group - 
create_form-header()
 Create form header - 
create_form-footer()
 Create form footer - 
create_form-text()
 Create form text -

Examples:

SCSS

.custom-form-group-name {
  @include create_form-group();
}

.custom-form-label-name {
  @include create_form-label();
}

HTML

<form action="#">
  <div class="custom-form-group-name">
    <label class="custom-form-label-name">Name</label>
    <input type="text" class="text-input" placeholder="Your name">
  </div>
</form>

Setting form layout config

The framework has common input settings and individual text input settings. Settings are in _source/config.scss file.

Settings list

Name Description Values
$form-label_color Label text color color
$form-label_font-family Label text font family font-family css property values
$form-label_font-size Label text font size font-size css property values
$form-label_line-height Label text line height line-height css property values
$form-label_font-style Label text font style font-style css property values
$form-label_font-weight Label text font weight font-weight css property values
